# 206. 反转链表

> 链接：[206. 反转链表](https://leetcode-cn.com/problems/reverse-linked-list/)  
> 标签：链表  
> 简介：反转一个单链表。

## 题解 1 - java

- 编辑时间：2020.2.13
- 执行用时：0ms
- 内存消耗：37.3MB
- 编程语言：java
- 解法介绍：创建链表，循环头部插入。

```java
class Solution {
    public ListNode reverseList(ListNode head) {
        if (head == null || head.next == null)
			return head;
		ListNode newHead=null;
		while(head!=null) {
			ListNode tmp=head.next;
			head.next=newHead;
			newHead=head;
			head=tmp;
		}
		return newHead;
    }
}
```
