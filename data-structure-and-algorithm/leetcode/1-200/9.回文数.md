# 9. 回文数

> 链接：[9. 回文数](https://leetcode-cn.com/problems/palindrome-number/)
> 标签：数学
> 简介：给你一个字符串 s 和一个字符规律 p，请你来实现一个支持 '.' 和 '\*' 的正则表达式匹配。

## 题解 1 - javascript

- 编辑时间：2020.4.12
- 执行用时：248ms
- 内存消耗：46.1MB
- 编程语言：javascript
- 解法介绍：转换成字符串进行翻转判断。

```javascript
/**
 * @param {number} x
 * @return {boolean}
 */
var isPalindrome = (x) =>
  x.toString() === x.toString().split("").reverse().join("");
```
